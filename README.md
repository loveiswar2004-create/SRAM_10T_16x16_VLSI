# SRAM 10T 16x16 Bit Memory Design

## Overview

This project presents the design and simulation of a 256-bit SRAM memory array using a 10-transistor (10T) cell architecture.

## Specifications

* Architecture: 16 x 16 array
* Total Capacity: 256 bits
* Cell Type: 10T SRAM
* Tools: Cadence Virtuoso, Spectre

## Included Files

* Report (PDF)
* Presentation (PPTX)
* Exported schematics
* Simulation results

## Key Analysis

* Read/Write operation
* Static Noise Margin (SNM)
* Power consumption
* Access delay
